# Selenide + Allure + JUnit

## To run tests:

Run test with chrome `mvn clean test -P chrome`

Run test with firefox `mvn clean test -P firefox`

## To see a report:

Run `mvn allure:serve`

It will automatically open `http://localhost:8080` in your browser
